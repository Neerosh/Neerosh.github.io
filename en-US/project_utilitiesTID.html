Utilities TID

Set of versatile tools and utilities to speed up my work as a Clarion desktop programmer. App developed using C# desktop using .NET 6.0 and SQLite.

Required .NET Desktop Runtime 6.0.3 ou SDK 6.0.3

GitHub Repository

Resources

Files

Add, change or delete file filters. Use filters to copy, move or delete duplicated files. Generate or validate file hashes.

Code Snippets

Add, change or delete repetitive code snippets. Use the registered codes to quickly copy to the clipboard.

Converter

Convert times and dates to the standard format used by the Clarion language. Find out the day of week of any date in Clarion format.

Processes

List or search processes, check the user responsible for each process and end process with one click.

Locks

Check if files are open by any process or shared folder. Find out which user is responsible for the process or shared folder.

SQL Scripts

Generate predefined scripts using specific paths on your machine.

Watcher

Watch over files or folders in real time, choose the events to be notified: name change, last access, etc...
