Utilities TID

Um conjunto de ferramentas versáteis para uso diário. Aplicação desenvolvida em C# utilizando .NET 6.0 e SQLite.

Requer .NET Desktop Runtime 6.0.3 ou SDK 6.0.3

Projeto no GitHub

Recursos

Arquivos

Inclua, altere ou exclua filtros de arquivos, utilize dos filtros para copiar, mover ou excluir arquivos duplicados. Gere ou valide hashes de arquivos.

Códigos

Inclua, altere ou exclua trechos repetitivos de código, utilize os códigos cadastrados para rapidamente copiar para a área de transfêrencia.

Conversor

Converta horários e datas para o formato padrão utilizado pela linguagem Clarion. Descubra o dia da semana de qualquer data em formato Clarion.

Processos

Liste ou pesquise processos, verifique o usuário responsável por cada processo e finalize processo com um clique.

Travas

Verifique se arquivos estão em aberto por algum processo ou compartilhamento e descrubra o usuário responsavel pelo processo ou compartilhamento.

SQL Scripts

Gere scripts pré-definidos utilizando caminhos especificos da sua máquina.

Vigia

Vigie arquivos ou pastas em tempo real, escolha os eventos a serem notificados: alteração de nome, ultimo acesso, etc..
